# **App Name**: ISRA EthniConnect

## Core Features:

- Intelligent Dashboard: Visualizes total sales, customer volume, and live stock tracking with real-time analytics for the shop manager.
- Customer Insights Tool: Generates personalized shopping suggestions for returning customers based on their purchase history logic.
- Omni-Channel Inventory: Dynamic management of saree, churidar, and kurti stock levels with automated low-stock alerts and cloud sync.
- Rapid Checkout System: Mobile-optimized invoice flow designed to create detailed receipts with INR currency formatting in under 30 seconds.
- High-Fidelity PDF Engine: Engineered PDF generator producing professional A4 brand-ready invoices with precise margins and shop branding.
- Direct WhatsApp Linker: One-tap sharing tool that integrates with the WhatsApp API to deliver PDF receipts or download links instantly.
- Data Storage Layer: Utilizes a secure cloud-hosted database for persistent customer records, inventory levels, and historical billing data.

## Style Guidelines:

- The design adopts a High-End Fashion Boutique aesthetic. Primary color: Midnight Navy (#151F2F) for professional, authoritative headings. Background: Soft Porcelain (#F9FAFC) for a crisp, spacious boutique feel.
- Accent: Cerulean Steel (#448FA3) for interactive elements and CTA buttons, providing sophisticated contrast against the primary dark navy.
- Headline and secondary text pairing: 'Playfair' for an elegant, high-end fashion vibe in headings, paired with 'PT Sans' for highly legible body copy and transactional data.
- Minimalist, thin-stroke linear icons inspired by high-fashion labels to denote product categories like sarees and churidars.
- A mobile-first, modular card-based layout featuring oversized touch targets to ensure speed and accuracy in a busy retail environment.
- Micro-transitions such as smooth fades when switching between dashboard views and subtle slide-ins for invoice confirmations.
